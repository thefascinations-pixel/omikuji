# Omikuji

A mobile-first web app inspired by the experience of drawing an omikuji at a Japanese shrine.

## Features

- Tap or shake to draw a fortune
- Calm paper-reveal animation with shrine-inspired visuals
- Natural Japanese fortune content from a 200-slip dataset
- Share or copy the current result
- Tie away unlucky fortunes such as `末吉` and `凶`
- Restore the last drawn fortune from local storage

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
